### Opção 1: Email do EAS Build (RECOMENDADO)

1. Verifique seu email: **grapanerari@hotmail.com**
2. Procure por um email do EAS Build com o assunto:
   ```
   "Build completed: @graciani/eduplay-games"
   ```
3. Clique no link para baixar o arquivo AAB
4. Salve o arquivo em local seguro

### Opção 2: Acessar Diretamente

1. Vá para: **https://expo.dev/accounts/graciani/projects/eduplay-games/builds**
2. Faça login com: **grapanerari@hotmail.com**
3. Procure pelo build mais recente
4. Clique em "Download" para obter o arquivo AAB

### Opção 3: Usar EAS CLI

```bash
eas build:list
eas build:download --id <build-id>
```
