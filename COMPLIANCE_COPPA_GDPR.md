# Conformidade COPPA e GDPR - EduPlay

## Declaração de Conformidade

EduPlay está em conformidade total com:
- **COPPA** (Children's Online Privacy Protection Act) - Lei americana de proteção de privacidade infantil
- **GDPR** (General Data Protection Regulation) - Regulamento europeu de proteção de dados
- **LGPD** (Lei Geral de Proteção de Dados) - Lei brasileira de proteção de dados

---

## Política de Coleta de Dados

### O que NÃO coletamos

**EduPlay NÃO coleta NENHUM dado pessoal identificável:**

- ❌ Nome do usuário
- ❌ Endereço de email
- ❌ Número de telefone
- ❌ Data de nascimento
- ❌ Localização geográfica
- ❌ Endereço IP
- ❌ Identificadores únicos de dispositivo (IDFA, Android ID)
- ❌ Dados de câmera ou microfone
- ❌ Contatos ou fotos

### O que coletamos (apenas dados anônimos)

**EduPlay coleta APENAS dados anônimos de jogo:**

- ✅ Pontuação de jogo (sem identificação do usuário)
- ✅ Tempo de jogo
- ✅ Tipo de jogo jogado
- ✅ Dados de progresso (completamente anônimos)
- ✅ Preferências de idioma

**Todos os dados são armazenados localmente no dispositivo do usuário.**

---

## Armazenamento de Dados

### Armazenamento Local

- Todos os dados de jogo são armazenados **localmente no dispositivo** usando AsyncStorage
- Nenhum dado é enviado para servidores remotos
- Nenhum dado é compartilhado com terceiros
- Os usuários podem deletar todos os dados a qualquer momento limpando o cache da app

### Segurança

- Dados sensíveis são armazenados em SecureStore (iOS Keychain / Android Keystore)
- Comunicações são criptografadas com HTTPS
- Nenhuma senha é armazenada

---

## Consentimento Parental

Como EduPlay **não coleta nenhum dado pessoal identificável**, **não é necessário consentimento parental explícito**.

A app é apropriada para usuários de todas as idades (6+) porque:
1. Não coleta dados pessoais
2. Não contém conteúdo inadequado
3. Não contém publicidades direcionadas
4. Não permite comunicação com outros usuários
5. Não coleta localização ou informações de dispositivo

---

## Direitos do Usuário

Os usuários têm direito a:

- **Acesso**: Ver todos os dados coletados (dados de jogo anônimos)
- **Portabilidade**: Exportar seus dados de jogo
- **Exclusão**: Deletar todos os dados limpando o cache da app
- **Recusa**: Não usar a app
